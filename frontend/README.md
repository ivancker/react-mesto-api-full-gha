## Проектная работа 12
"Регистрация и авторизация"

Сервис Mesto - это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Использованные технологии:**
1. HTML
2. CSS
3. JavaScript
4. NPM
5. Webpack
6. REST API
7. React JS
8. JSX

**Реализация:**
* Grid Layout
* Flexbox
* Медиазапросы
* Nested БЭМ
* ОПП
* React
